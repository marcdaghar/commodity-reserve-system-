# Commodity Reserve Systems and Countercyclical Storage

This repository contains the code for the paper "Commodity Reserve Systems and Countercyclical Storage: A Formal Framework for Monetary Stability" by Marc Daghar.

## Overview

The framework includes three main components:
1. CRD price stabilization with floor-ceiling controls
2. Yusufian countercyclical storage rule
3. Multi-objective optimization for storage location
